Observer Pattern

The observer pattern is being used for subscribing to beers. A user can subscribe and unsubscribe from beers and then they will be notified once the beer has changed in price. The Beer model has a notifyObservers method in which all of its observers are sent an email on price change.



Template Pattern

The template pattern is being used for filtering. A user can select to filter by either beers or stores. Once those have been selected, the filter method filters properly by using the user's selections.
